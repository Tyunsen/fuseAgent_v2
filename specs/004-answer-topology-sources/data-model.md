# Data Model: ApeRAG Answer Topology And Sources

## Answer Topology Block

- **Purpose**: The existing Mermaid-backed `流程拓扑` area inside an answer.
- **Source of truth**: Existing answer markdown content.
- **Key attributes**:
  - `mermaidSource`: The Mermaid text generated by the current answer flow.
  - `renderState`: Visual graph, raw data view, or render-failed fallback.
  - `viewportState`: Current zoom/pan state while viewing the graph.

## Source Drawer State

- **Purpose**: UI state for opening and inspecting answer sources through the ApeRAG-style drawer.
- **Key attributes**:
  - `isOpen`: Whether the source drawer is visible.
  - `expandedRowIds`: Which source rows are expanded.
  - `activeRowIds`: Which source rows are highlighted or most recently opened.

## Source Reference Row

- **Purpose**: One source entry in the drawer corresponding to one supporting passage.
- **Key attributes**:
  - `id`: Stable row identifier.
  - `documentName`: Display name for the source document.
  - `documentId`: Optional source document identifier.
  - `pageIdx`: Optional page index.
  - `sectionLabel`: Optional heading or section hint derived from the source text.
  - `paragraphPrecise`: Whether the locator is exact enough to present as precise.
  - `snippet`: Short preview text shown in the collapsed row.
  - `text`: Full passage shown when expanded.

## Source Locator

- **Purpose**: The user-facing location marker shown for a source row.
- **Possible components**:
  - Page number
  - Section or heading label
  - Exact vs approximate precision marker
- **Rule**: Must use the best trustworthy locator available and must not imply exact paragraph precision when the data does not support it.
