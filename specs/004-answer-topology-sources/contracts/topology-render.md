# Contract: Topology Render

## Scope

This contract defines the user-visible behavior of the existing answer
`流程拓扑` block.

## Contract

1. The answer continues to use the existing Mermaid topology content generated by the current answer flow.
2. The visible graph presentation is upgraded toward the approved MiroFish style without requiring a second graph module.
3. The graph view must still allow switching to the raw topology data view.
4. If the visual graph cannot be rendered, the answer still remains usable and the raw topology data remains accessible.
5. The graph renderer change must not require changes to the user's answer submission flow.
